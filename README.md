## 🚀 Features

- **Responsive Design**: Optimized for different screen sizes, ensuring a seamless reading experience on desktops, tablets, and mobile devices.
- **Comprehensive News Layout**: Reproduces the complex grid and list layouts used by Hindustan Times to display top news, trending stories, and category-specific content.
- **Interactive Navigation**: Features a detailed "Explore" menu and secondary navigation for various news categories like Lifestyle, Trending, and World News.
- **Dynamic Content Sections**: Includes dedicated sections for Cricket, Entertainment, Health, Astrology, and Education, each with its own unique styling.
- **Live Updates UI**: Mimics the look and feel of real-time news updates with "Live Story" indicators and time-stamped articles.

## 🛠️ Technologies Used

- **HTML5**: Semantic elements were used to structure the news content, improving accessibility and SEO.
- **CSS3**: Advanced styling including Flexbox/Grid for complex layouts, custom animations for hover states, and media queries for responsiveness.
- **Google Fonts**: Integrated modern typography to match the professional look of a news publication.

## 📦 How to Run Locally

To view this project locally on your machine:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Het-Roy/Hindustan-Times.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd Hindustan-Times/hindustanTimes
    ```
3.  **Open in Browser**:
    Simply open the `index.html` file in your preferred web browser (Chrome, Firefox, Edge, etc.).
